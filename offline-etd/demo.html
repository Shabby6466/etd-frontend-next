How to Use

  1. Click "Launch Application" to start
  2. Use any of the demo user credentials to login
  3. Fill in the citizen information form
  4. Upload a photo (any image file under 10MB)
  5. Optionally capture fingerprint (requires SecuGen device)
  6. Submit the form to create an application
Launch Application

Technical Information

Framework: Vanilla JavaScript, HTML5, CSS3

Styling: Tailwind CSS

Biometric: SecuGen WebAPI

Storage: Local browser storage

Compatibility: Modern browsers (Chrome, Firefox, Safari, Edge)
